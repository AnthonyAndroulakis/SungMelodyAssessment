# Sung Melody Transcription
Sung Melody Transcription in Python, this is a vastly-more-efficient version of my SMM.m

## Requirements:
1) .wav file of a sung melody, any length
2) python3
3) python modules:
    - scipy
    - more_itertools
    - parselmouth
    - numpy

## How to run:
`import SMM`       
`notePitches,noteLengths = SMM.SMM(filename)`
